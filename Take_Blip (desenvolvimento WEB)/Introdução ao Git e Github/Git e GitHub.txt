Comandos básicos para um bom desempenho no terminal:
	dir -> mostra os arquivos
	cd -> vai pra uma pasta
	mkdir -> cria a pasta
	cls -> clear
	echo -> print('texto')
	cd.. -> sai da pasta
	del -> deleta arquivos e se for uma pasta ele apaga o qque ta dentro dela
	-a -> mostra os arquivos escondidos
	rmdir -> deleta pastas ex: rmdir "pasta(sem as aspas)" /S /Q(esse /S e /Q se tiver coisa dentro dela.
